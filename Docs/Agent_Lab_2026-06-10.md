# Agent Lab — the AI space (One Conductor, deep)

Spawn discipline **personas** and **teams**, hand them a goal + sources, and an
**orchestrator** runs them. Every run feeds **Hermes** so the agents improve.
Built on the existing substrate (`callLlm` per-user router + Hermes learning
engine), not from scratch.

Verified in-sandbox: `tsc -b` green + **75/75 targeted vitest** (orchestrator 8,
store 6, Conductor 19, hidden-widgets 5, serviceLaunch 5, UserContext 18,
providerSSR 14). The live LLM runs are your Mac check — they need an API key set
in Settings → API Keys.

---

## How a team takes on a task (your orchestrator model)

1. **Decompose** — the Orchestrator splits the goal into a task list per member.
2. **Execute** — each specialist completes its tasks, with Hermes few-shot from
   its own past successes injected as context.
3. **Verify** — each output is checked against the **sources you provided**;
   unsupported claims are flagged `[UNVERIFIED]`.
4. **Merge** — the Orchestrator merges the verified outputs into one final product.

A single persona can also run solo (execute → verify).

## Self-improvement (Hermes)

Every persona run is recorded (`recordRun`) and future runs pull the top-K
similar past **successes** as few-shot context (`relevantPastRuns`). 👍/👎 on the
final product calls `rateRun`, biasing what the agents reuse. All per-user.

## Personas + teams

Seven built-in discipline specialists (Orchestrator, Researcher, Legal Analyst,
Engineer, Data Analyst, Comms Writer, Strategist) + three teams (Research Squad,
Deal Desk, Build Team). You can create/edit your own personas (name, discipline,
system prompt) and teams (pick members). Built-ins are read-only; customs persist
per-user and sync via One Save.

## The fold (heavy)

The standalone agent widgets — **Stella, Hydra, Two Brains, Synthesis, Hive,
Builder Agents** — are now **hidden from the sidebar by default** (one-time,
reversible). The Agent Lab is the single door. Any of them is re-addable from
**"+ Add widget"** if you miss it. ARA Console (quick chat) and the memory
widgets (Honcho, Cognitive M Network) stay.

## Open it

Sidebar → AI Tools → **Agent Lab**, or say **"spawn a research team"** / **"open
agent lab"** to ⌘K / ARA.

## Honest scope notes

- Orchestration is the model you specified (orchestrator task-lists → complete →
  verify-against-sources → merge). It's sequential per member; parallel execution
  is a later optimization.
- Verification is LLM-judged against the pasted sources (not a live web/citation
  fetch). Wiring it to the real FactCheck/web tools is a natural follow-up.
- Hermes here is run-memory + tool-weighting (no model fine-tuning — that's an
  Electron/GPU concern, as the existing Hermes docs note).
